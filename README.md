## Install dependencies

```bash
# Install smaug
$ brew tap ereborstudios/tap
$ brew install smaug

# Install dependencies
$ smaug dragonruby install ~/Downloads/dragonruby-gtk-macos.zip
$ smaug install
```

## Run the game
```bash
$ smaug run
```
